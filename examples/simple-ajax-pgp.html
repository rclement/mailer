Mailer: simple AJAX example with PGP encryption

Demonstrate how to use mailer using a simple form and an AJAX request, using Vue.js.
If enabled, PGP encryption of the e-mail will be performed server-side automatically.
An optional PGP public key can be attached with the PGP-encrypted e-mail: this way the receiver can also respond with an end-to-end encrypted message!
